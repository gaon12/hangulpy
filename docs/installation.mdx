## 설치

```bash
pip install hangulpy
```

## 업그레이드

```bash
pip install -U hangulpy
```

## 가져오기

```python
from hangulpy import josa, romanize
```

## 개발용 설치

저장소를 직접 열고 테스트까지 실행하려면 아래처럼 설치하면 됩니다.

```bash
pip install -e .[dev]
python -m pytest -o addopts=''
```
