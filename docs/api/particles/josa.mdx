title: 조사 붙이기 (josa)

---

단어 끝의 받침 유무를 판단해 적절한 조사를 붙입니다.

```python
josa(word: str, particle: str) -> str
```

## josa_pick

단어에 붙일 조사 형태만 반환합니다.

```python
josa_pick(word: str, particle: str) -> str
```

## 예시

```python
from hangulpy import josa, josa_pick

print(josa("사과", "을/를"))  # 사과를
print(josa("책", "이/가"))  # 책이
print(josa("나무", "으로/로"))  # 나무로
print(josa("서울", "으로/로"))  # 서울로
print(josa("3", "이/가"))  # 3이
print(josa_pick("사과", "이/가"))  # 가
```

## 지원 범위

현재 구현은 74개의 조사 쌍을 지원합니다. 대표적으로 아래 조합을 자주 씁니다.

| 조사 쌍 | 예시 | 결과 |
| --- | --- | --- |
| `을/를` | `josa("사과", "을/를")` | `사과를` |
| `이/가` | `josa("책", "이/가")` | `책이` |
| `은/는` | `josa("사과", "은/는")` | `사과는` |
| `와/과` | `josa("책", "와/과")` | `책과` |
| `으로/로` | `josa("나무", "으로/로")` | `나무로` |
| `이라/라` | `josa("사과", "이라/라")` | `사과라` |
| `이나/나` | `josa("책", "이나/나")` | `책이나` |
| `이에요/예요` | `josa("사과", "이에요/예요")` | `사과예요` |
| `이라고/라고` | `josa("책", "이라고/라고")` | `책이라고` |

`으로/로` 계열 조사는 `ㄹ` 받침 뒤에서 `로` 를 선택합니다. 예를 들어 `서울`, `길` 은 각각 `서울로`, `길로` 가 됩니다.
