## 핵심 변경

- `docs/` 기반 문서 구조를 정리하고, 함수별 API 설명을 보강했습니다.
- `hangul_contains`, `HangulSearcher`, `number_to_hangul`의 동작을 보정했습니다.
- 국어의 로마자 표기법 규정을 기준으로 `romanize`와 `Romanizer`를 크게 개선했습니다.

## 로마자 표기 개선

- 발음 변화 반영 범위를 넓혀 비음화, 유음화, 구개음화, `ㅎ` 결합, `ㄴ` 첨가 등을 더 정확히 처리합니다.
- `mode`, `capitalize`, `disambiguate`, `name_hyphen`, `admin_omit_suffix` 옵션을 추가해 인명, 행정 구역, 붙임표 표기까지 다룰 수 있게 했습니다.

## 문서 변경

- 외부 위키 의존을 제거하고 공식 문서 사이트 기준으로 문서를 다시 구성했습니다.
- 함수마다 기본값, 생략 시 동작, 예시 코드, 결과값을 함께 볼 수 있게 정리했습니다.
